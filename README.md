# InfnetFood

Aplicativo mobile desenvolvido em React Native com Expo para pedidos e delivery de refeições, como parte da disciplina Desenvolvimento Mobile com React Native.

## Funcionalidades

- Login com validação básica
- Listagem de categorias
- Listagem de produtos
- Tela de detalhes do produto
- Carrinho de compras
- Checkout
- Perfil do usuário
- Pedidos
- Mapa de restaurantes
- Detalhes do restaurante
- Tema claro e escuro
- Notificações simuladas
- Integração com API de CEP (ViaCEP)

## Tecnologias utilizadas

- React Native
- Expo
- React Navigation
- Expo Notifications
- React Native WebView
- ViaCEP API
